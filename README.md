# Pengenalan

<b>Sistem Pengurusan Pusat Jagaan Anak Yatim (e-PJAY)</b> merupakan sistem maklumat pengurusan berasaskan web bagi membantu pihak kakitangan dalam meningkatkan kecekapan dan keberkesanan untuk menguruskan maklumat mengenai proses permohonan kemasukan ke rumah anak yatim, pengurusan maklumat penghuni dan pengurusan kewangan. Antara pengguna bagi sistem ini ialah pihak kakitangan dan penjaga kanak-kanak. Pembangunan sistem ini menggunakan kerangka kerja <b>Laravel</b> dengan pangkalan data <b>MySQL</b> manakala model air terjun digunakan sebagai metodologi perisian.

# Senarai Fungsi Sistem

Penjaga

-   Menghantar Permohonan Kemasukan
-   Menyemak Status Permohonan
-   Membuat Turun Surat Tawaran
-   Melihat Maklumat Anak Jagaan
-   Melihat Paparan Dashboard

Pihak Kakitangan

-   Membuat Pengesahan Permohonan
-   Mengurus Maklumat Anak Jagaan
-   Memuat Turun Senarai Anak Jagaan (PDF, Excel, CSV)
-   Mengurus Maklumat Pendapatan/Perbelanjaan
-   Melihat Paparan Dashboard
